## 🚀 **DEPLOYMENT INSTRUCTIONS**

### **1. Firebase Setup**
```bash

dart pub global activate flutterfire_cli

flutterfire configure

```

### **2. Build Commands**
```bash
# Get dependencies
flutter pub get

# Build for Android
flutter build apk --release

# Build for iOS
flutter build ios --release

```
